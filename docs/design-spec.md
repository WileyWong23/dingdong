# 叮咚 DingDong - UI 设计规范

## 1. 设计理念

贯彻 **Apple 产品设计哲学**：少即是多，内容为王，细节至上。

### 核心原则

| 原则 | 说明 |
|------|------|
| **克制** | 每个像素都有意义，拒绝装饰性元素 |
| **留白** | 大量呼吸空间，让内容自然呈现 |
| **层次** | 通过字号、字重、颜色建立清晰的信息层级 |
| **动效** | 微妙的过渡动画，增强直觉感知，不炫技 |
| **一致** | 统一的圆角、间距、色彩系统 |

## 2. 视觉参考产品

### 2.1 一级参考（设计理念）

| 产品 | 参考点 |
|------|--------|
| **Apple Reminders** | 原生感、干净的列表、SF 字体运用 |
| **Things 3** | 任务管理 UI 的天花板，留白与层级的极致 |
| **Linear** | 现代 SaaS 设计标杆，暗色主题的精致感 |

### 2.2 二级参考（交互细节）

| 产品 | 参考点 |
|------|--------|
| **Structured** | 时间轴可视化、日视图的优雅 |
| **Fantastical** | 自然语言输入创建事件 |
| **Apple Health** | 卡片式数据展示、色彩运用 |

## 3. 色彩系统

### 3.1 主色

```
背景：    #FAFAFA  (近乎白色，温暖不刺眼)
表面：    #FFFFFF  (卡片/弹窗)
主文字：  #1D1D1F  (Apple 标准黑)
次要文字：#6E6E73  (Apple 标准灰)
分隔线：  #D2D2D7  (极淡灰线)
```

### 3.2 强调色

```
主强调：  #007AFF  (Apple 蓝，用于按钮/链接)
成功：    #34C759  (绿色，启用状态)
警告：    #FF9500  (橙色，待处理)
危险：    #FF3B30  (红色，删除/过期)
```

### 3.3 暗色模式

```
背景：    #000000
表面：    #1C1C1E
主文字：  #FFFFFF
次要文字：#8E8E93
分隔线：  #38383A
```

## 4. 字体系统

### 4.1 字体选择

Web 端优先使用系统字体栈，保证原生感：

```css
font-family: -apple-system, BlinkMacSystemFont, "SF Pro Display", 
             "SF Pro Text", "Helvetica Neue", Helvetica, Arial, sans-serif;
```

### 4.2 字号层级

| 用途 | 字号 | 字重 | 行高 | 示例 |
|------|------|------|------|------|
| 页面大标题 | 34px | Bold (700) | 1.2 | "提醒" |
| 区域标题 | 22px | Semibold (600) | 1.3 | "今天的提醒" |
| 列表标题 | 17px | Medium (500) | 1.4 | 提醒名称 |
| 正文 | 15px | Regular (400) | 1.5 | 提醒详情 |
| 辅助文字 | 13px | Regular (400) | 1.4 | 时间/状态 |
| 标签 | 11px | Semibold (600) | 1.2 | 类型标签 |

## 5. 间距系统

基于 **8px 网格**：

```
xs:  4px   (图标与文字间距)
sm:  8px   (相关元素间距)
md: 16px   (段落/卡片内边距)
lg: 24px   (区块间距)
xl: 32px   (页面边距)
2xl: 48px  (大区块分隔)
```

## 8. 动效规范

| 交互 | 动效 | 时长 |
|------|------|------|
| 弹出面板 | 从底部滑入 + 淡入 | 300ms ease-out |
| 关闭面板 | 向下滑出 + 淡出 | 250ms ease-in |
| 列表项增删 | 高度动画 + 淡入淡出 | 200ms |
| 完成提醒 | 圆圈填充 + 文字划线 | 300ms |
| 按钮点击 | scale(0.97) 回弹 | 100ms |
| 页面切换 | 淡入淡出 | 200ms |

## 9. 响应式适配

| 断点 | 布局 |
|------|------|
| < 640px (手机) | 单列，全宽卡片 |
| 640-1024px (平板) | 居中内容区 max-width 600px |
| > 1024px (桌面) | 居中内容区 max-width 720px |

## 10. PWA 安装引导

首次访问时展示优雅的安装引导：
- 不使用丑陋的浏览器弹窗
- 自定义底部提示卡片："添加到主屏幕，获得最佳体验"
- 提供分步图示引导
- 一次性展示，用户关闭后不再打扰
